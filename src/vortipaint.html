Abstract

I wanted to create a meditative drawing tool with a simple UI. Vortex Paint is the result of my experiments with p5.js, a library that enables one to use the Processing programming language in a JavaScript setting. As a very visual and iterative project, I included a lot of example images generated by me, as well as other users during test runs of the site among art students at the Maryland Institute College of Arts, as well as friends and family.

Design Challenge

Create an unconventional drawing tool that invokes a meditative user experience using p5.js, a JavaScript/web-friendly adaptation of Processing.
