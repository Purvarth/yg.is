A concise visualization method for ancestral relations with d3.js

4 weeks · UI designer, web developer · 2018

Visit Live Site

Abstract

Family trees by nature have always seemed very mathematical to me, almost like fractal trees. I chose to use the d3.js javaScript library in order to design and develop an open-source web application that visualizes familial bonds in the form of a sunburst chart, which resembles a chopped down trunk,. Trunks explores the fractal relationships of ancestry through the visualization of a large database of family trees scraped from Wikipedia.

Design Challenge

Develop a simple visualization method for genealogical data that presents ancestral relations concisely, thoroughly and in a usable manner.
